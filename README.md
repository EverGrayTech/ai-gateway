# EverGray Tech: AI Gateway

`@evergraytech/ai-gateway` is a centralized AI gateway that securely proxies model requests, enforces usage limits, and enables a zero-setup default experience alongside BYOK.

## Installation

```bash
pnpm add @evergraytech/ai-gateway
```
